# 创建自由实践文件

当用户需要在自由实践目录下创建新的草稿/伪代码文件时，按照以下步骤操作。

// turbo-all

## 步骤

1. 向用户确认文件名称（如果未提供），例如：`二分查找思路`、`dp状态转移`、`图论模板` 等。

2. 在 `d:\竞赛真题\自由实践\` 下创建 `.txt` 文件，文件名即为用户提供的名称。

3. 文件内容模板：

   **如果用户提供了具体内容**，直接写入用户提供的内容。

   **如果用户只给了主题**，使用如下模板：
   ```
   {文件名称}
   ============
   日期：{当前日期 YYYY-MM-DD}

   思路：


   伪代码：


   复杂度分析：

   ```

4. 创建完成后告知用户文件路径。

## 注意事项

- 文件扩展名统一使用 `.txt`，确保不会触发编译器或 IntelliSense 检查。
- 文件名使用中文即可，简洁明了。
- 不需要创建子目录，所有文件平铺在 `自由实践/` 下。
